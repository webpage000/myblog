<h1 align="center">MyBlog</h1>

>webpage000的个人博客项目

## 简介
**MyBlog** 是将来我的个人博客项目，将来会将自己的学习进度以及心得发布在上面，以便作为回顾，因此后续会将其部署到服务器上，并对其进行升级以及维护。在此也希望各位能帮忙发现问题并指出，谢谢！

## 项目结构

```
├─ myblog
│  ├─ advice // 全局异常处理
│  ├─ controller // 控制器
│  ├─ entity // 实体
│  ├─ mapper // xml 映射接口
│  ├─ service // 业务执行类
│  ├─ utils // 工具类
│  └─ MyblogApplication 
```
